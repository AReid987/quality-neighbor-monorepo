# Product Roadmap

## Phase 1: Core Platform Foundation & Onboarding (MVP Focus)

**Goal:** Establish the fundamental technical infrastructure and enable initial user/business onboarding with a positive, secure, and compliant first impression.
**Success Criteria:** Launch of the platform with core features, 100+ resident sign-ups, and 5+ businesses onboarded.

### Features

- [x] **Secure Authentication & Authorization (AuthN/AuthZ):** Securely register for an account and log in/out. `[L]`
- [x] **High-Conversion Resident Landing Page:** Easily understand what Quality Neighbor offers and sign up for the free newsletter. `[M]`
- [ ] **Basic User Profiles (Residents):** Create and manage a basic profile. `[M]`
- [x] **Essential Legal Compliance (Privacy & Terms):** Provide clear legal documentation to our users. `[S]`

### Dependencies

- None

## Phase 2: Resident-to-Resident Exchange (MVP Focus)

**Goal:** Enable core community interactions by facilitating the exchange of services, tools, and skills among residents.
**Success Criteria:** 20+ active listings and 10+ successful exchanges within the first month.

### Features

- [ ] **Service/Tool/Skill Listing Creation:** Offer a service/tool/skill or request help. `[L]`
- [ ] **Browse & Discover Listings:** Search and filter available service/tool/skill listings. `[M]`
- [ ] **Secure In-App Messaging for Exchange:** Send a private message to another resident about a listing. `[L]`
- [ ] **Trust-Building Identity Verification (MVP):** Verify the identity of users offering services. `[M]`

### Dependencies

- Phase 1 completion

## Phase 3: Local Business Onboarding & Ad Management (MVP Focus)

**Goal:** Provide local businesses with a seamless way to advertise and track their performance, fostering a valuable local ecosystem.
**Success Criteria:** 10+ businesses running active ad campaigns.

### Features

- [ ] **Business Owner Account Setup:** Register my business and link it to my user account. `[M]`
- [ ] **Business Owner Dashboard (MVP):** View an overview of my advertising activity and performance. `[L]`
- [ ] **Ad Campaign Submission & Agent Support:** Submit my ad creative and copy, and get assistance from the Quality Neighbor agent team. `[L]`
- [ ] **Basic Local Business Directory (Resident Facing):** Discover local businesses featured on Quality Neighbor. `[M]`

### Dependencies

- Phase 1 completion

## Phase 4: Content & Collaboration

**Goal:** Empower the marketing team with content management capabilities and enhance internal collaboration.
**Success Criteria:** Successful integration of a headless CMS and real-time collaborative editor.

### Features

- [ ] **Integrate Prismic:** Allow marketing team members to add and manage content. `[L]`
- [ ] **Integrate Liveblocks:** Add a Notion-like collaborative editor to the internal app. `[L]`
- [ ] **Project Management Features:** Add basic project management tools to the internal app. `[XL]`

### Dependencies

- Phase 3 completion

## Phase 5: Analytics & Personalization

**Goal:** Implement robust user analytics and personalization features.
**Success Criteria:** Successful integration of analytics tools and a noticeable improvement in user engagement metrics.

### Features

- [ ] **Integrate PostHog:** Add product analytics to the user-facing qn-platform. `[L]`
- [ ] **Integrate Mixpanel/OpenPanel:** Add event tracking to the qn-dashboard internal app. `[L]`
- [ ] **Implement Robust Memory:** Implement a robust memory solution for personalization and user behavior data collection. `[XL]`

### Dependencies

- Phase 4 completion
